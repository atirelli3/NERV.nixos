Create the three enable-only service modules: pipewire.nix, bluetooth.nix, and printing.nix. Each wraps its existing flat module body inside a lib.mkIf cfg.enable guard behind a single nerv.*.enable option.

The critical correctness concern is avahi ownership: printing.nix must own avahi.enable = true directly (not rely on pipewire.nix), and bluetooth.nix must also own it independently — both safe because NixOS merges duplicate bool assignments with logical OR.
Output: Three new files in modules/services/; original flat modules in modules/ untouched.
