  <name>Task 1: Clone NERV.nixos and copy refined structure</name>
  <files></files>
  <action>
    **Step A — Clone the NERV.nixos repo:**

    ```bash
    git clone git@github.com:atirelli3/NERV.nixos.git /tmp/nerv-nixos
    ```

    If the clone fails (SSH auth, empty repo error), check SSH key is loaded: `ssh-add -l`. If the repo is truly empty (no commits), git clone may fail — in that case, initialize manually:
    ```bash
    mkdir /tmp/nerv-nixos
    cd /tmp/nerv-nixos
    git init
    git remote add origin git@github.com:atirelli3/NERV.nixos.git
    ```

    **Step B — Copy refined files using rsync:**

    ```bash
    # Copy flake.nix
    cp /home/demon/Developments/test-nerv.nixos/flake.nix /tmp/nerv-nixos/flake.nix

    # Copy home/
    mkdir -p /tmp/nerv-nixos/home
    cp /home/demon/Developments/test-nerv.nixos/home/default.nix /tmp/nerv-nixos/home/default.nix

    # Copy modules/ recursively (excludes nothing — flat modules/*.nix were deleted in Plan 01)
    cp -r /home/demon/Developments/test-nerv.nixos/modules /tmp/nerv-nixos/

    # Copy hosts/ — select files only (NOT nixos-base/ subdirectory)
    mkdir -p /tmp/nerv-nixos/hosts
    cp /home/demon/Developments/test-nerv.nixos/hosts/configuration.nix /tmp/nerv-nixos/hosts/configuration.nix
    cp /home/demon/Developments/test-nerv.nixos/hosts/disko-configuration.nix /tmp/nerv-nixos/hosts/disko-configuration.nix
    cp /home/demon/Developments/test-nerv.nixos/hosts/nixos-base/hardware-configuration.nix /tmp/nerv-nixos/hosts/hardware-configuration.nix

    # Copy README.md if it exists
    [ -f /home/demon/Developments/test-nerv.nixos/README.md ] && cp /home/demon/Developments/test-nerv.nixos/README.md /tmp/nerv-nixos/README.md
    ```

    **Step C — Verify structure:**

    ```bash
    find /tmp/nerv-nixos -not -path '*/.git/*' | sort
    ```

    Expected output should show: flake.nix, home/default.nix, hosts/configuration.nix, hosts/disko-configuration.nix, hosts/hardware-configuration.nix, modules/default.nix, modules/services/*.nix, modules/system/*.nix

    Verify NO flat modules/*.nix files (other than default.nix) are present:
    ```bash
    ls /tmp/nerv-nixos/modules/*.nix
    ```
    Should return only: modules/default.nix

    **Step D — Commit and push:**

    ```bash
    cd /tmp/nerv-nixos
    git add flake.nix home/ hosts/ modules/
    [ -f README.md ] && git add README.md
    git commit -m "feat: initial NERV.nixos release — v1.0

    Graduated from test-nerv.nixos development repo.
    Structure: modules/system/ + modules/services/ with nerv.* options API.
    Three inline profiles: hostProfile, serverProfile, vmProfile.
    Full impermanence support for server profile (mode = full)."
    git push -u origin main
    ```

    After push, verify:
    ```bash
    git -C /tmp/nerv-nixos log --oneline -3
    ```
