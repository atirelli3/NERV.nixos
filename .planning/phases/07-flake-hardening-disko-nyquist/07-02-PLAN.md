<objective>
Wire `disko` as a proper flake input; add `disko.nixosModules.disko` and `./hosts/nixos-base/disko-configuration.nix` to `nixosConfigurations.nixos-base`; fix the ESP mount options in `disko-configuration.nix` to use `fmask=0077 dmask=0077`; remove the `lib.mkForce` fileSystems and swapDevices overrides from `configuration.nix`.

Purpose: Closes tech debt item D from the v1.0 audit. The `lib.mkForce` overrides were a workaround for the missing disko module — once disko is wired, the overrides are actively harmful (they suppress the security-relevant mount options declared in `disko-configuration.nix`). After this plan, disk layout has a single source of truth: `disko-configuration.nix`.

Output: `flake.nix` with disko input + module, `disko-configuration.nix` with corrected ESP mount options, `configuration.nix` with fileSystems/swapDevices blocks removed and header updated.
</objective>
