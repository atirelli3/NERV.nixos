**Comment header:**
```
# modules/system/impermanence.nix
#
# Purpose  : Selective per-directory tmpfs mounts — /tmp, /var/tmp, and user volatile dirs.
# Model    : Root (/) stays on disk (ext4). Specific dirs are mounted as tmpfs at boot.
#            The upstream nixos-community/impermanence module is NOT used.
# Options  : nerv.impermanence.enable, persistPath, extraDirs, users
# Defaults : enable = false; persistPath = "/persist" (reserved); extraDirs = []; users = {}
# Override : lib.mkForce on any fileSystems.* entry, or disable impermanence entirely.
```

**Options to declare** (all under `options.nerv.impermanence`):
- `enable = lib.mkEnableOption "selective per-directory tmpfs impermanence"`
- `persistPath` — `lib.mkOption` with `type = lib.types.str`, `default = "/persist"`, description: `"Persistence base path. Reserved for forward compatibility; unused in the selective tmpfs model."`
- `extraDirs` — `lib.mkOption` with `type = lib.types.listOf lib.types.str`, `default = []`, description: `"Additional absolute system paths to mount as tmpfs. Additive to the defaults (/tmp, /var/tmp)."`, example: `[ "/var/cache/app" ]`
- `users` — `lib.mkOption` with `type = lib.types.attrsOf (lib.types.attrsOf lib.types.str)`, `default = {}`, description: `"Per-user additional tmpfs mounts. Keys are usernames; values are attrsets mapping absolute path to size string (e.g. \"4G\", \"25%\")."`, example: `{ demon0 = { "/home/demon0/Videos" = "8G"; }; }`

**Config block** (`config = lib.mkIf cfg.enable { ... }`):

1. **IMPL-02 assertion** — sbctl safety check. Use `lib.mkIf config.nerv.secureboot.enable` to make the assertion conditional (only relevant when secureboot is also enabled):
```nix
assertions = lib.optional config.nerv.secureboot.enable {
  assertion =
    let
      allPaths = cfg.extraDirs
        ++ lib.concatLists (lib.mapAttrsToList (_user: pathMap: lib.attrNames pathMap) cfg.users);
      isSbctlPath = p: p == "/var/lib/sbctl"
        || lib.hasPrefix "/var/lib/sbctl" p
        || lib.hasPrefix p "/var/lib/sbctl";
    in
      !(lib.any isSbctlPath allPaths);
  message = "nerv: /var/lib/sbctl is in impermanence tmpfs paths — this would wipe Secure Boot keys on every reboot. Remove it from impermanence configuration.";
};
```

2. **Default system tmpfs mounts** — `/tmp` and `/var/tmp` as fileSystems entries:
```nix
fileSystems."/tmp" = {
  device  = "tmpfs";
  fsType  = "tmpfs";
  options = [ "size=25%" "mode=1777" "nosuid" "nodev" ];
};
fileSystems."/var/tmp" = {
  device  = "tmpfs";
  fsType  = "tmpfs";
  options = [ "size=25%" "mode=1777" "nosuid" "nodev" ];
};
```

3. **extraDirs mounts** — Use `lib.mkMerge` + `builtins.listToAttrs` to generate fileSystems entries for each path in `cfg.extraDirs`. Each entry: `device = "tmpfs"`, `fsType = "tmpfs"`, `options = [ "size=25%" "nosuid" "nodev" ]`.

4. **Default per-user mounts** (Desktop + Downloads for each user in `config.nerv.primaryUser`) — Use `lib.mkMerge` + `map` over `config.nerv.primaryUser`. For each user, generate two fileSystems entries: `/home/${user}/Desktop` and `/home/${user}/Downloads`. Options: `[ "size=25%" "nosuid" "nodev" ]`. Also add `systemd.tmpfiles.rules` entries to create the directories before mount: `"d /home/${user}/Desktop 0755 ${user} users -"` and `"d /home/${user}/Downloads 0755 ${user} users -"`. This prevents boot failure if the mount point does not exist on disk (Pitfall 5 from RESEARCH.md).

5. **Per-user custom mounts** from `cfg.users` — Use `lib.mkMerge` + `lib.concatLists` + `lib.mapAttrsToList` to iterate `cfg.users`. For each `(username, pathMap)` pair, iterate `pathMap` to generate fileSystems entries at each `path` with `options = [ "size=${size}" "nosuid" "nodev" ]`. Also add corresponding `systemd.tmpfiles.rules` `d` entries for each path.

Use `lib.mkMerge` at the top level of the config block to combine all the fileSystems attrsets cleanly.
